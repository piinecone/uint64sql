# uint64sql

SQL-safe uint64s in go

## Install

Run `go get github.com/piinecone/uint64sql`

## Usage

```go
package main

import (
    "fmt"
    "github.com/piinecone/uint64sql"
)

func main() {
    num := uint64sql.New(1234567890)
}
```
